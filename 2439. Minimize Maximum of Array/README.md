# 2439. Minimize Maximum of Array

## Problem

You are given a 0-indexed array `nums` consisting of non-negative integers.

In one operation, you can choose an index `i` such that `1 <= i < nums.length` and `nums[i] > 0`, then decrease `nums[i]` by `1` and increase `nums[i - 1]` by `1`.

Return the minimum possible value of the maximum element of `nums` after performing any number of operations.

---

## Example 1

Input:

nums = [3,7,1,6]

Output:

5

Explanation:

One optimal sequence of operations can make the maximum value in the array equal to 5.

---

## Example 2

Input:

nums = [10,1]

Output:

10

Explanation:

It is not possible to make the maximum element smaller than 10.

---

## Constraints

- 2 <= nums.length <= 10^5
- 0 <= nums[i] <= 10^9
