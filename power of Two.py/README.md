# Power of Two

## Problem Statement
Given an integer `n`, determine whether it is a power of two.

A number is a power of two if there exists an integer `x` such that:

n = 2^x

Return `true` if `n` is a power of two; otherwise, return `false`.

## Example 1

Input:
n = 1

Output:
true

Explanation:
2^0 = 1

## Example 2

Input:
n = 16

Output:
true

Explanation:
2^4 = 16

## Example 3

Input:
n = 3

Output:
false

## Constraints

- -2^31 <= n <= 2^31 - 1

## Task

Determine whether the given integer `n` is a power of two and return the result.
