# Rotate Array by One Place

## Problem Statement
Given an array of integers, rotate the array by one position to the right.

## Example 1

Input:
arr = [1, 2, 3, 4, 5]

Output:
[5, 1, 2, 3, 4]

## Example 2

Input:
arr = [10, 20, 30, 40]

Output:
[40, 10, 20, 30]

## Constraints

- 1 <= arr.length <= 10^5
- -10^9 <= arr[i] <= 10^9

## Task
Rotate the given array by one position to the right and return the modified array.
