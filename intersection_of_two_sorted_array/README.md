# Intersection of Two Sorted Arrays

## Problem

Given two sorted arrays, find their intersection.

The intersection contains all elements that appear in both arrays in sorted order.

---

## Example 1

Input:

arr1 = [1,2,3,4,5]

arr2 = [1,2,5,7,9]

Output:

[1,2,5]

---

## Example 2

Input:

arr1 = [1,2,3]

arr2 = [4,5,6]

Output:

[]

---

## Constraints

- 1 <= arr1.length, arr2.length <= 10^5
- Arrays are sorted in non-decreasing order
- 1 <= arr1[i], arr2[i] <= 10^9
