# Ugly Number

## Problem Statement
An ugly number is a positive integer whose prime factors are limited to `2`, `3`, and `5`.

Given an integer `n`, return `true` if `n` is an ugly number. Otherwise, return `false`.

## Example 1

Input:
n = 6

Output:
true

Explanation:
6 = 2 × 3

## Example 2

Input:
n = 8

Output:
true

Explanation:
8 = 2 × 2 × 2

## Example 3

Input:
n = 14

Output:
false

Explanation:
14 = 2 × 7, and 7 is not an allowed prime factor.

## Constraints

- -2^31 <= n <= 2^31 - 1

## Task

Determine whether the given integer `n` is an ugly number and return the result.
