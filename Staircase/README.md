# Staircase

## Problem Statement
Given a positive integer `n`, print a staircase of size `n` using the `#` symbol. The staircase should be right-aligned.

## Example

Input:
n = 4

Output:
   #
  ##
 ###
####

## Example 2

Input:
n = 6

Output:
     #
    ##
   ###
  ####
 #####
######

## Constraints

- 1 <= n <= 100

## Task
Print a right-aligned staircase of height `n` using the `#` symbol.
