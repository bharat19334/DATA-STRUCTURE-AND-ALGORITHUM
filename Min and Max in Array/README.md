# Min and Max in Array

## Problem

Given an array `arr[]`, find the minimum and maximum elements present in the array.

Return both values.

---

## Example 1

Input:

arr = [3, 2, 1, 56, 10000, 167]

Output:

1 10000

---

## Example 2

Input:

arr = [1, 345, 234, 21, 56789]

Output:

1 56789

---

## Constraints

- 1 <= arr.length <= 10^5
- 1 <= arr[i] <= 10^12
