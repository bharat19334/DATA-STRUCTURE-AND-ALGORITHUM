# Missing Element

## Problem

Given an array containing `n-1` distinct numbers taken from the range `1` to `n`, find the missing number.

---

## Example 1

Input:

arr = [1,2,4,5]

Output:

3

---

## Example 2

Input:

arr = [2,3,1,5]

Output:

4

---

## Constraints

- 1 <= n <= 10^5
- Array contains distinct integers
- Exactly one number is missing
- Numbers are in the range [1, n]
