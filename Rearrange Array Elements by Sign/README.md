# Rearrange Array Elements by Sign

## Problem

You are given an array `nums` consisting of an equal number of positive and negative integers.

Rearrange the array such that positive and negative integers alternate, starting with a positive integer.

The relative order of positive and negative integers should be preserved.

---

## Example 1

Input:

nums = [3,1,-2,-5,2,-4]

Output:

[3,-2,1,-5,2,-4]

---

## Example 2

Input:

nums = [-1,1]

Output:

[1,-1]

---

## Constraints

- 2 <= nums.length <= 2 × 10^5
- nums.length is even
- Equal number of positive and negative integers
- -10^5 <= nums[i] <= 10^5
