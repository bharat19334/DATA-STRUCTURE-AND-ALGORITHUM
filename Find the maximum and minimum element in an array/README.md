# Find the Maximum and Minimum Element in an Array

## Problem

Given an array `arr[]` of size `n`, find the minimum and maximum elements present in the array.

Return both the minimum and maximum values.

---

## Example 1

Input:

arr = [3, 2, 1, 56, 10000, 167]

Output:

min = 1

max = 10000

---

## Example 2

Input:

arr = [1, 345, 234, 21, 56789]

Output:

min = 1

max = 56789

---

## Example 3

Input:

arr = [10]

Output:

min = 10

max = 10

---

## Constraints

- 1 <= n <= 10^5
- 1 <= arr[i] <= 10^12
