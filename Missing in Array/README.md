# Missing in Array

## Problem

Given an array of size `n-1` containing distinct integers from `1` to `n`, find the missing number.

---

## Example 1

Input:

arr = [1,2,3,5]

Output:

4

---

## Example 2

Input:

arr = [6,1,2,8,3,4,7,10,5]

Output:

9

---

## Constraints

- 1 <= n <= 10^6
- 1 <= arr[i] <= n
